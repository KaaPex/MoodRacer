# MoodRacer
Client-server racer game

# Test Site
http://kaapex.github.io/MoodRacer

# How to use
Install dependencies:
```shell
npm install
```
Start dev server:
```shell
npm run start
```

Client side changes set to hot update, you doon't need run dev-server every time.

# Testing via mocha
```shell
npm run test
```
